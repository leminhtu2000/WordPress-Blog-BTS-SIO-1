# Choix techniques

## CMS
Le CMS WordPress a été choisi pour les raisons suivantes :
- Sécurité et mises à jour régulières
- Bonne performance et rapidité
- Optimisation du référencement naturel (SEO)
- Large choix de thèmes et plugins gratuits

## Thème
Un thème gratuit (Variation) a été utilisé et personnalisé afin de respecter la charte graphique du projet.

## Plugins utilisés
- Plugin de sécurité
- Plugin SEO
- Plugin pour le sauvegarde

## Hébergement
Le site est hébergé sur une plateforme compatible WordPress permettant un accès public au blog.
